# Git Lesson

## リモートリポジトリとローカルリポジトリとはそれぞれ何でしょうか？

リモートリポジトリ→GitHub上にある、ファイルやディレクトリの保存場所。
ローカルリポジトリ→自分のPCにある、ファイルやディレクトリの保存場所。

## プッシュとマージの違いは何でしょうか？

pushは変更内容をGitHubへデータを送ること
mergeは別のブランチで行った変更内容をデフォルトのブランチに反映させること

## コミットとプッシュの違い

commitは名前をつけてローカル上にセーブ
pushはローカルの内容をGitHubに送ること

## コミットのメッセージはどのように書いてあげるのが最適でしょうか？

変更内容を明確にして書く

## ローカルでマージするフローと、プルリクエストでマージするフローの違いは何でしょうか？

GitHub上でmergeを行うか、ターミナルでmergeを行うか
→プルリクエストを作成することで反映内容を確認した後、mergeすることができるため、
作業ミスの防止となる

## コンフリクトを起こしてしまった場合、どう対処すべきですか？
採用する変更内容を決め、修正を行い、add commit pushを行う
